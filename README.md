tijekids_web_Herrera

Proyecto final del curso donde implementé un sistema web hecho con Django para gestionar turnos y reseñas.

La idea principal fue practicar modelos, formularios, templates, URLs, vistas y autenticación básica con login/logout.

Funcionalidades principales

Crear turnos indicándo cliente, servicio, fecha y duración.

Ver listado de turnos cargados.

Agregar reseñas a los turnos.

Ver todas las reseñas publicadas.

Detalle individual de cada turno.

Sistema de login y logout con usuario registrado.

Tecnologías usadas

Python 3

Django 5

Bootstrap 5
