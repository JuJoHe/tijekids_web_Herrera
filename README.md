# tijekids_web_Herrera

Proyecto Final Django - TijeKids Web (Juan José Herrera)

Aplicación web desarrollada en Django para gestionar turnos de la peluquería infantil TijeKids y permitir que los clientes puedan dejar reseñas y opiniones sobre el servicio. Incluye:
- Administración de servicios
- Reserva de turnos
- Reseñas de clientes visibles públicamente
- Sección About
- Formularios con validaciones
- Templates con Bootstrap 5

Proyecto final integrador para el curso de Django.
